Aprendizajes / Dificultades enfrentadas

Aprendizajes del equipo

Análisis profundo del caso de negocio

Aprendimos a realizar un análisis detallado de requerimientos y necesidades del cliente, conectando problemas con soluciones técnicas.

Diseño de arquitectura de microservicios

Comprendimos las ventajas de un diseño modular y escalable, planificando la separación de responsabilidades en servicios independientes.

Aplicación de metodología Scrum

Implementamos exitosamente prácticas ágiles como sprints, dailies, y refinamiento del backlog para mantener una planificación adaptativa.

Selección de tecnologías apropiadas

Evaluamos y elegimos el stack tecnológico adecuado (Java, Spring, Oracle, AWS, JWT) en base a requerimientos y escalabilidad futura.

Definición clara de requisitos y reglas

Reconocimos la importancia de documentar claramente requisitos, roles y reglas de negocio antes de comenzar el desarrollo.

Autoaprendizaje y adaptación técnica

Identificamos recursos clave para el autoaprendizaje y organizamos un plan de capacitación para cerrar brechas de conocimiento.

Dificultades a superar

Complejidad de microservicios

El diseño e implementación de una arquitectura de microservicios por primera vez representa un desafío de aprendizaje para el equipo.

Estrategia: Estudiar patrones de diseño y casos de éxito

Curva de aprendizaje tecnológica

Familiarizarse con Spring Boot, AWS y JWT requiere tiempo de aprendizaje que podría afectar la velocidad inicial de desarrollo.

Estrategia: Documentación Oficial, Cursos y Consultas a Docentes.

Integración entre componentes

Asegurar la correcta comunicación entre microservicios y con la base de datos requiere una planificación detallada.

Estrategia: Definir contratos de API claros y pruebas automatizadas

Gestión del tiempo y alcance

Mantenerse dentro de los plazos de dos semanas por sprint y evitar el scope creep es un desafío constante.

Estrategia: Priorización estricta y refinamiento continuo del backlog

Nivel de confianza del equipo por área

"Cada obstáculo es una oportunidad para aprender y mejorar nuestras habilidades como equipo."
