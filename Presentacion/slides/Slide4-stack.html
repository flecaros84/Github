Tecnologías que se utilizarán

Stack Tecnológico

Backend

Java con Spring Boot (Arquitectura de Microservicios)

Facilita escalabilidad e independencia entre módulos

Base de Datos

Oracle Database (desplegada en AWS RDS)

Base de datos relacional potente y escalable

Frontend

HTML, CSS, JavaScript (con React o Vue)

Interfaces responsivas y experiencia de usuario moderna

Seguridad

Spring Security con JSON Web Tokens (JWT)

Autenticación y autorización robusta para APIs

API Gateway

Spring Cloud Gateway

Enrutamiento y filtrado centralizado de peticiones

Otras tecnologías clave:

AWS EC2/RDS

Justificación de la Arquitectura

  • Los microservicios permiten desarrollo, despliegue y escalamiento independiente
  • Cada equipo puede trabajar en servicios específicos sin afectar otros
  • El API Gateway centraliza autenticación y facilita la comunicación con clientes
  • Oracle ofrece alta confiabilidad para datos críticos del negocio
